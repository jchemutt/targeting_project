About

Targeting Tools is a set of tools available in Web. The toolbox has three tools: Land Similarity, Land Suitability, and Land Statistics.

Tools Features

Land Similarity

Estimates the potential for out-scaling using socio-ecological characterization and similarity analysis.

Land Suitability

Matches suitability criteria with spatial databases to create suitability maps.

Land Statistics

Calculates zonal statistics, e.g., total area and mean covered by population.
