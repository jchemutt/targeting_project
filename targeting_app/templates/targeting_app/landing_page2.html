About

Targeting Tools identifies land suitable for various crops using geospatial data.

Tools Features

Land Similarity

Identifies locations with similar conditions to known suitable areas.

Land Suitability

Analyzes spatial data to determine land suitability for crops.

Land Statistics

Calculates zonal statistics for land areas.

Web Application

Runs geoprocessing tools from any browser, with preprocessed input data.

Get Started

Desktop Application

Runs on ArcGIS Desktop 10.3 or later, requires no internet connection.
